# Recharts Dashboard

A simple and responsive data visualization dashboard built using **React**, **TypeScript**, **Recharts**, and **Tailwind CSS**.

## Features

* Line, Bar, Area, and Pie charts
* Reusable chart card components
* Responsive grid layout
* Clean UI with Tailwind CSS
* Data table for raw data display

## Tech Stack

* React (Vite)
* TypeScript
* Recharts
* Tailwind CSS

## Run Locally

```bash
git clone https://github.com/your-username/recharts-dashboard.git
cd recharts-dashboard
npm install
npm run dev
```
